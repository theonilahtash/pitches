# Pitch-App
## By
### **Theonilah Owali**

## Description

This application allows users to write a one minute pitch in  different  categories. They can also view other peoples pitches and be able to comment on them.


### categories include:

* Product Pitch
* Promotion Pitch
* Business Pitch
* Technology Pitch
* Career Pitch

## User Stories

As a user I would like:
* to view the different categories
* to see the pitches other people have posted
* to submit a pitch in any category
* to comment on the different pitches and leave feedback
* to vote on the pitch and give it a downvote or upvote

## Specifications
| Behavior        | Input           | Outcome  |
| ------------- |:-------------:| -----:|
| Register to be a user | Your email : theo@.com <br> Username : theo <br> Password : th34eo | New user is registered |
| Log in | Your email : theo@.com<br> Password : th34eo | Signed in |
| Display pitch categories | N/A | List of various pitch categories |
| See pitches from selected category | **Click** a category | Directed to a page with a list of pitches from the selected category |
| Create a pitch | **Click Add Pitch** | An authenticated user is directed to a page with a form where the user can create and submit a pitch |
| See a pitch | **Click** on a pitch | A user is directed to a page containing the pitch, its comments and its votes |
| Comment on a pitch | **Click Comment** | An authenticated user is directed to a page with a form where the user can create and submit a comment on a pitch |

* open terminal and go to the project folder, run $ ./start.sh
